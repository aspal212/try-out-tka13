# API Documentation

## Authentication

- POST /api/auth/login
- POST /api/auth/register
- POST /api/auth/logout

## Exams

- GET /api/exams
- POST /api/exams
- GET /api/exams/:id
- PUT /api/exams/:id
- DELETE /api/exams/:id

## Questions

- GET /api/questions
- POST /api/questions
- PUT /api/questions/:id
- DELETE /api/questions/:id

## Users

- GET /api/users
- PUT /api/users/:id
